# rdc-knowledge-base

Red Door Collaborative knowledge base. Claude conversation outputs, project context, design system, and reference materials.

Modeled after [Joe Garvin's PARA-style claude-knowledge-base](https://github.com/joe-garvin/claude-knowledge-base).

## Structure

- **`projects/`** - Active client and internal work projects
- **`areas/`** - Ongoing responsibilities and recurring contexts
- **`resources/`** - Reference material, frameworks, prompt libraries, research
- **`docs/`** - Documentation, guides, and the canonical RDC design system
- **`archive/`** - Completed projects and historical context
- **`meta/`** - System notes, conventions, changelog, repository guidelines

## Active Projects

- [file-taxonomy](projects/file-taxonomy/) - RDC File Taxonomy & Organization SOP

## Conventions

- File names: lowercase, hyphenated, descriptive
- Each substantive file opens with a header comment indicating creation date and originating session
- All HTML deliverables use [the RDC design system](docs/rdc-design-system.html)

See [`meta/conventions.md`](meta/conventions.md) for the full guide and [`CLAUDE.md`](CLAUDE.md) for the context loaded into Claude sessions.
