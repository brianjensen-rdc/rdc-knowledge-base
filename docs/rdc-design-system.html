Typography

Primary typeface: Suisse Int'l (Swiss Typefaces). Digital fallback: Proxima Nova. Web stand-in: DM Sans via Google Fonts. Mono: DM Mono. Weights: 300, 400, 500, 600, 700. Max 3 weights per page.

Alerts

Warning: Coral alert. Use for warnings or risks.
Info: Blue info alert. General informational callouts.
Success: Green success alert. Confirmation or locked decisions.

Logo & Assets

RDC logo lives in Box: Red Door File Database -> Red Door Guidelines and Assets -> RDC 2025 Branding. Mark min 24px digital / 0.35in print; full lockup 120px / 1.5in. Clear space = height of the "r".

Do & Don't

Do

  • Lead with Coral for CTAs
  • Use Squid for dark surfaces
  • Sentence case headings (labels excepted)
  • Use approved Box files for the logo
  • Candid, human, professional photography

Don't

  • Use Mist as a primary background
  • Put Coral text on Wine (contrast fails)
  • Recreate or screenshot the logo
  • Use drop shadows on the logo
  • Use staged stock photography

Usage

All HTML deliverables in this repo MUST use this design system. Copy the :root CSS variables and component classes. No off-brand colors without design-lead approval.
